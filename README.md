# Proyecto de especialización en Ingeniería de software

## Descripción

Este proyecto es parte de la especialización en Ingeniería de software y tiene como objetivo desarrollar una aplicación web que permita a los usuarios evaluar sus conocimientos en fundamentos de programación, identificar sus debilidades técnicas y recibir recomendaciones personalizadas para mejorar sus habilidades de programación.
