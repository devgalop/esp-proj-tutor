# Algoritmos de clasificación

Son una categoría de algoritmos de aprendizaje supervisado cuyo objetivo es utilizar las características de entrada para predecir una etiqueta de salida que pertenece a una clase específica. Estos algoritmos se utilizan para resolver problemas de clasificación, donde el objetivo es asignar cada instancia a una categoría o clase.

Su función principal es predecir la etiqueta de clase a la que pertenece una instancia dada sus características de entrada. Para lograr esto, los algoritmos de clasificación aprenden patrones y relaciones en los datos de entrenamiento para luego aplicarlos a nuevos datos.

Es importante tener en cuenta que los algoritmos de clasificación tienen las siguientes características:

- **Variable de salida discreta**: La etiqueta de salida es una variable categórica que representa la clase a la que pertenece cada instancia. Por ejemplo, en un problema de clasificación de correo electrónico, las clases podrían ser "spam" y "no spam".
- **Depende de datos etiquetados**: Los algoritmos de clasificación requieren un conjunto de datos etiquetado para entrenar el modelo. Cada ejemplo de entrenamiento debe incluir tanto las características de entrada como la etiqueta de clase correspondiente.
- **Se deben establecer fronteras de decisión**: Los algoritmos de clasificación establecen fronteras de decisión en el espacio de características para separar las diferentes clases. Estas fronteras pueden ser lineales o no lineales, dependiendo del algoritmo utilizado.
